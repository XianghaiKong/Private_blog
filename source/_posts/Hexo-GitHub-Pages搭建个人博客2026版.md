## 阶段二：配置专属独立域名

博客搭建在 GitHub Pages 上后，默认的域名通常是 `用户名.github.io`。为了让博客看起来更正式、更有辨识度，我决定为它绑定一个专属域名。

🔗 **参考教程**：[《GitHub Pages 绑定自定义域名》](https://zhuanlan.zhihu.com/p/873902706)

**核心操作**：
1. **购买域名**：我直接在阿里云购买了一个以 `.top` 结尾的域名（性价比极高）。
2. **DNS 解析配置**：在阿里云的控制台中，添加 `CNAME` 记录，将其解析指向我自己的 GitHub Pages 地址。
3. **GitHub 配置**：在 GitHub 博客仓库的 `Settings -> Pages -> Custom domain` 中填入刚买的域名，并开启 `Enforce HTTPS`。稍等片刻待 DNS 刷新后，就能通过您的专属独立域名访问博客了！

## 阶段三：博客主题美化与自定义

原生的 Hexo 界面比较极简，为了提升阅读体验和博客的颜值，主题美化是最后也是最折腾人的一个环节。

🔗 **参考教程**：[《Hexo 博客主题深度美化指南》](https://www.cnblogs.com/qianxiaohan/p/19032755)

**主要的美化方向包括但不限于**：
- 安装并替换第三方的主题（如经典的 NexT 主题等）。
- 配置侧边栏头像、个人信息及社交账号链接。
- 修改博客的背景、字体配色，调整文章正文内容的排版样式。
- 配置了专属的网站浏览器标签页小图标（Favicon）。

经过这一系列的折腾与打磨，一个排版优雅、拥有专属独立域名的 2026 版个人博客就正式落成啦！
